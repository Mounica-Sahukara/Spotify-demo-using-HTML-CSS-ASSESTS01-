# Spotify UI Demo

This is a Spotify-inspired UI demo built using HTML and CSS. 
It focuses on layout and design only and does not play real music.

## Project Status
- Currently under development (about 90% completed)
- Layout and design mostly done

## Features
- Spotify-like user interface
- Responsive layout using HTML and CSS

## Technologies Used
- HTML
- CSS

## Note
This is a front-end demo project for learning purposes. 
No backend or music playback functionality is included.
Future updates may improve or change the UI design, but no backend features are planned.
